# Summary

## Abstract

This project serves as a practical application of the concepts learned in the **Human-Machine Interface (HMI)** course. The objective is to design and implement a comprehensive **Human Resources Management System (HRMS)** that addresses the complex needs of modern organizations in managing their workforce.

The project follows the **Rational Unified Process (RUP)** methodology, ensuring a structured approach to development from requirements analysis to implementation. We utilized **UML (Unified Modeling Language)** for system modeling, providing clear visual representations of the system's structure and behavior.

The application is built using **Flutter** for the frontend, ensuring a cross-platform (Windows, Web, Mobile) and responsive user interface, connected to a **REST API** backend for robust data management. Key features include employee lifecycle management, organizational structure handling (Departments/Bodies), and bilingual support, all presented through an ergonomic and user-friendly interface.

**Keywords:** HRMS, Flutter, RUP, UML, HMI, Cross-platform, Management System.

## Résumé

Ce projet constitue une application pratique des concepts acquis lors du module **Interface Homme-Machine (IHM)**. L'objectif est de concevoir et réaliser un système complet de **Gestion des Ressources Humaines (GRH)** qui répond aux besoins complexes des organisations modernes.

Le projet suit la méthodologie **RUP (Rational Unified Process)**, assurant une approche structurée du développement, de l'analyse des besoins jusqu'à l'implémentation. Nous avons utilisé le langage **UML** pour la modélisation du système.

L'application est développée avec **Flutter** pour garantir une interface utilisateur ergonomique, responsive et multiplateforme, connectée à une **REST API** pour la gestion des données.

**Mots-clés:** GRH, Flutter, RUP, UML, IHM, Multiplateforme.
